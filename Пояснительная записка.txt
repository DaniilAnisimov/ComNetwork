1.Название проекта.
ComNetwork

2.Авторы проекта.
Анисимов Даниил Олегович и Козлов Алексей Олегович

3.Описание идеи.
Сайт является коллективной сетью,
где каждый зарегистрированный пользователь может выкладывать посты,
комментировать и оценивать чужие посты.

4.Описание реализации.
Сайт сделан на фреймворке flask. Для работы с бд присутствует api.
Запросы к api выполняются с помощью библиотеки requests
